# Verse Intel

## What is Verse Intel?

Verse Intel is a web application for Star Citizen that allows players to search, organize, and manage player information.

This project started as a way for me to learn JavaScript, but my long-term goal is to turn it into a complete ecosystem that includes a web application, desktop application, and Discord bot.

---

## Why am I building it?

- Learn full-stack web development
- Learn software engineering
- Build a portfolio project
- Solve a problem for the Star Citizen community

---

## Current Goal

Finish the MVP.

Current features:

- Search players
- Add players
- Store data with localStorage

---

## Future Ideas

- Player search through an API
- User accounts
- Cloud database
- Organization management
- Electron desktop app
- Discord bot
